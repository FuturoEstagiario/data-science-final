## Pipeline ETL — Detalhado

### EXTRACT — 4 fontes, 3 métodos

| Fonte | Método | Função | Detalhe |
|---|---|---|---|
| `smartphones.csv` | Arquivo local | `extrair_csv()` | 499 smartphones, colunas em EUR |
| `open.er-api.com` | HTTP GET (REST API) | `extrair_taxa_cambio()` | Taxa EUR→BRL ao vivo; fallback R$ 5,90 |
| Wikipedia | Web Scraping (BeautifulSoup) | `extrair_wikipedia()` | Telefones mais vendidos — `wikitable` |
| `smartphones.db` | SQLite (`sqlite3`) | `extrair_sqlite()` | Re-leitura do banco gerado na fase Load |

### TRANSFORM

**Limpeza e enriquecimento (`limpar_dados`):**
- RAM nulos (150 registros): tentativa de extração via **regex** `(\d+)/\d+GB` no campo `Smartphone`; valores restantes preenchidos com mediana
- Normalização de espaços em `Brand`, `Model`, `Color` com `re.compile(r'\s+')` e `.sub(' ', x).strip()`
- Conversão de preços EUR → BRL multiplicando `Final Price × taxa_brl`; resultado em `Preco_BRL`
- Categorização com `pd.cut` em 3 faixas: **Budget** (≤ R$ 1.500) · **Mid-range** (R$ 1.501–4.000) · **Premium** (> R$ 4.000)
- Cruzamento com marcas da Wikipedia: coluna `Top_Vendedor` (0/1)

**Modelo ML (`treinar_modelo`):**
- Features: `RAM`, `Storage`, `Brand_enc` (LabelEncoder), `Free_enc` (binário)
- Target: `Categoria` (3 classes)
- Pipeline scikit-learn: `StandardScaler` → `RandomForestClassifier(n_estimators=100, random_state=42)`
- Avaliação: `cross_validate` 5-fold CV + holdout 30%
- Resultados obtidos: **CV 78,75%** · **Teste 82,00%**

### LOAD — 4 saídas

| Saída | Biblioteca | Conteúdo |
|---|---|---|
| `relatorio.xlsx` | openpyxl | 3 abas: Dados (499 linhas), Resumo por Marca + BarChart, Previsões ML |
| `relatorio.pdf` | reportlab | Resumo executivo, Top 10 por preço, Distribuição por categoria |
| `relatorio.html` | Python f-string | Tabela estática Top 20, Resumo por Marca, estilo inline |
| `view/js/data.js` | json.dump | Dados estruturados para o dashboard (meta, by_brand, by_category, by_storage, by_ram, phones) |

## Dashboard Interativo (`view/`)

**Tecnologias:** HTML5 · CSS3 (custom properties, CSS Grid) · Vanilla JS (ES2020) · Chart.js 4.4.0 (CDN)

### Estrutura Visual

- **Sidebar** fixa (240px): navegação por seção com ícones SVG inline; badge de acurácia ML no rodapé
- **Topbar** sticky: título, badge EUR/BRL ao vivo, timestamp de geração
- **5 seções scrolláveis:**

| Seção | Conteúdo |
|---|---|
| Visão Geral | 4 KPI cards (Total, Preço Médio, Marca Líder, ML %) |
| Mercado | Barras horizontais Top 10 marcas + Donut de categorias |
| Preços | Scatter RAM × Preço por categoria + Barras por Storage |
| Machine Learning | Barras CV vs Teste + Barras RAM × Preço Médio |
| Catálogo | Tabela interativa com busca, 4 filtros, sort por coluna, paginação |

### Paleta de Cores

| Token | Hex | Uso |
|---|---|---|
| `--sidebar-bg` | `#0b1629` | Sidebar (azul-petróleo escuro) |
| `--blue` | `#2563eb` | Primário, KPI azul, gráficos |
| `--green` (Budget) | `#10b981` | Badge Budget, KPI preço, gráfico ML |
| `--amber` (Mid-range) | `#f59e0b` | Badge Mid-range |
| `--purple` (Premium) | `#8b5cf6` | Badge Premium, gráfico RAM |
| `--card` | `#ffffff` | Cards brancos |
| `--bg` | `#f4f6fb` | Fundo geral (bege/cinza claro) |

### Interatividade JS

- **Filtros combinados:** busca por texto (brand + model), categoria, marca, RAM — reativos via `input`/`change`
- **Ordenação por coluna:** qualquer coluna da tabela; indicador `↑` / `↓` no `<th>`
- **Paginação com janela:** 15 linhas/página; botões `‹ 1 … n … › `
- **Nav suave:** `scrollIntoView({ behavior: 'smooth' })` + estado ativo no sidebar
